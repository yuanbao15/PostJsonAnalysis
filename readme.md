# Postman 请求json数据分析工具

@created by yuanbao at 202503

### 本工具作用：

用于对已有的postman json文件。进行分析处理，提取需要的功能点表格查询的数据生成excel。



### 实现：

1. 支持对多个业务模块文件json作处理
2. 每个业务模块生成一个sheet
3. form-data数据格式作转换
